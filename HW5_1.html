The Analysis Notebook

Visualization 1 – License Distribution by Type and Status

This first chart shows how many business licenses belong to each license type and how those licenses are divided by status. I used a bar chart because it clearly compares different categories. The x-axis shows the license types, and the y-axis shows the number of licenses in each group. I used color to show the different license statuses, which helps you see how each type is split between statuses like Active or Not Renewed. To prepare the data, I grouped it by both License Type and License Status and counted how many records were in each group.

Visualization 2 – Top 15 Counties by Number of Licenses (Interactive)

The second chart shows the top 15 counties with the highest number of business licenses. The x-axis shows the number of licenses, and the y-axis lists each county. I added click-to-select interactivity so a selected county is highlighted. Before plotting, I grouped the data by County and selected the 15 counties with the highest totals.

Interactivity Explanation

The interactivity in my second chart lets you click on a county to highlight it. This makes the visualization easier to read because your selected county stands out while the others fade into the background. It also helps compare values more clearly.
